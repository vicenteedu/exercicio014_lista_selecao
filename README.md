# Exercicio 14 da lista de seleção (Faculdade)

Este exercício tem como objetivo trabalhar estruturas condicionais em Java. O programa solicita ao usuário um número de 1 a 7, onde cada valor representa um dia da semana (1 para domingo, 2 para segunda-feira, e assim por diante).

Com base no número informado, o sistema exibe o dia correspondente por extenso. Caso o usuário digite um valor fora desse intervalo, o programa retorna uma mensagem de erro, garantindo a validação da entrada.

Desenvolvido utilizado JDK 21 e NetBeans 29
